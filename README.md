# Sales-analysis-using-python
The data has been generated randomly and some tools and libraries have been used like (pandas), (numpy), (metplotlib), (scipy.stats) , (seaborn)
Dataset
The dataset is synthetically generated inside this notebook (no external download needed), so it always runs end-to-end without internet access. It deliberately includes the kind of mess real data has — missing values, duplicate rows, inconsistent text casing, and a few outliers — so the cleaning steps are meaningful rather than cosmetic.



Project Outcomes

By the end of this notebook, we should be able to:

Diagnose and fix common real-world data quality issues
Perform structured EDA - Exploratory Data Analysis (univariate, bivariate, time-based)
Build an RFM (Recency, Frequency, Monetary) customer segmentation
Run a basic hypothesis test to support a business claim
Translate analysis into plain-language business recommendations
